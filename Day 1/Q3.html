JavaScript Datatypes

There are 8 basic data types in JavaScript

  • Numbers
    • - numbers of any kind: integer or floating-point, integers
  • Strings
    • - string may have zero or more characters
  • Boolean
    • - true/false
  • Objects
    • - for more complex data structures
  • Symbol
    • - for unique identifiers
  • bigint
    • - for integer numbers of arbitrary length
  • null
    • - for unknown values
  • undefined
    • - for unassigned values
